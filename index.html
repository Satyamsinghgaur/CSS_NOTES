CSS Introduction

CSS (Cascading Style Sheets) :-

CSS (Cascading Style Sheets) is a language used to style the visual appearance of HTML elements on a web page. It controls things like colors, fonts, spacing, and layout, allowing you to design and format your website's look and feel. In simple terms, CSS is what makes your webpage pretty!
CSS is a language that is used to describe the style of a document.
